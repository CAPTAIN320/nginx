Slider AI API Version 1

Generate a presentation

Example cURL request:


$ curl -s \
  -H "Authorization: Token <paste-your-api-key-here>" \
  https://app.sliderai.com/api/v1/presentation/generate \
  -d '{
      "prompt": "the porsche 911 sports car", // maximum of 100 characters
      "numOfSlides": 5 // between 3 and 20
  }'

The response will be a JSON object containing the presentationId:


{
  "status": 200,
  "presentationId": "68d43872-aed3-498f-8f37-18276437c12a"
}

Responses

https://app.sliderai.com/api/v1/presentation/generate

Get the presentation generation status

Example cURL request:

$ curl -s \
  -H "Authorization: Token <paste-your-api-key-here>" \
   https://app.sliderai.com/api/v1/presentation/status/<presentaionId-here>

The response will be a JSON:

{
  "presentationId": <the-presentation-id>,
  "status": "completed",
}

Responses

https://app.sliderai.com/api/v1/presentation/status/{presentationId}

Get the presentation file URL

Example cURL request:

$ curl -s \
  -H "Authorization: Token <paste-your-api-key-here>" \
   https://app.sliderai.com/api/v1/presentation/file/<presentaionId-here>

The response will be a JSON:

{
  "status": 200,
  "url": "https://sliderai.com/.../SliderAI-68d43872-aed3-498f-8f37-18276437c12a.pptx" // presentationId as the file name
}

Responses

https://app.sliderai.com/api/v1/presentation/file/{presentationId}
